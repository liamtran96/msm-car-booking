---
description: Create or manage pull requests
---

**Pre-PR checklist:**
- [ ] All tests pass
- [ ] Code compiles
- [ ] Linting passes
- [ ] Changeset added (if user-facing)

**Process:**
1. Run pre-PR checklist
2. Delegate to `code-reviewer` for final review
3. Generate PR description from commits
4. Create PR using `gh pr create`

$ARGUMENTS - Optional: PR title or target branch
